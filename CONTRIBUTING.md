# Contributing to Axiom GNU/Linux

Hey! Thanks for wanting to help build Axiom GNU/Linux. Whether you're a developer, Linux enthusiast, rice maker, or tester, you are super welcome here.

---

### 🚀 Ways You Can Help

1. **Test & Report Bugs**: Flash an ISO, try out Distropaks, or test `axm`. If something breaks, open a Bug Report on the relevant repo!
2. **Suggest Features**: Got an idea for a new Distropak tag or an `axm` subcommand? Open a Feature Request!
3. **Write Code & Scripts**: 
   - `axm` is written in C.
   - `axiom-tools` are written in bash/shell & C.
   - `distropaks` are Containerfiles & bash hooks.
4. **Improve Documentation & Website**: Fix typos, write wiki articles, or tweak CSS on `axiom-website`.

---

### 📜 Guidelines

- **Be friendly**: Be respectful, helpful, and constructive.
- **Join the Discord**: Chat directly on [Discord](https://discord.gg/9YHAZUZ4PX)!
